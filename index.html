Voidstar (Working Title)

'Voidstar' (or whatever I'm calling it this week) was my side project up until 2018. It had been in development on and off since August 2017, but has existed as a number of proof of concept's since ~2012, slowly increasing in scope and quality.

The main idea of this project is to produce an 'Asteroids-like' game featuring elements typical to much more recently released titles. The features chosen to be implemented in this project include: character customisation, competitive gameplay, AI opponents, and networked multiplayer. As of the current iteration, all features except multiplayer have been implemented to some extent.

At the present moment I have no plans to resume this project.

Made with: Unity 2017, C#
Contribution: All Gameplay, All Assets (except sound)

Entire Game on One Screen

This game was made over the course of one weekend in an attempt to immitate the conditions of a Game Jam. The theme was decided randomly from previous Ludum Dare Game Jams, and all game assets were produced within the assigned time period. Overall the game took 20 hours from theme to completion.

The core concept of the game is a puzzle platformer where abilities are unlocked over time, allowing previously visited rooms to be explored in new ways. The main gameplay ability to be unlocked allows the player to 'connect' two objects, allowing one to change position based on the movement of the other.

While this game is not neccessarily impressive from either a graphical or game mechanic perspective, it demonstrates a concept taken and developed to its completion.
